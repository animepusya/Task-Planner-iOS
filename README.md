# Task Planner

Task Planner is an iOS app focused on time-based planning, recurring tasks, and clear statistics to understand how you spend your time.

---

## 📲 Download on App Store

[![Download on the App Store](https://developer.apple.com/assets/elements/badges/download-on-the-app-store.svg)](https://apps.apple.com/us/app/task-planner-daily-schedule/id6762282029)

---

## ✨ Features

- 🗓️ Monthly calendar with selected-day task list  
- ⏱️ Time-based planning (not just task lists)  
- 🔁 Advanced recurring tasks (daily, weekly, monthly, custom rules)  
- 📊 Statistics (day / week / month / year) with insights  
- 🔔 Smart reminders and notifications  
- 📆 Apple Calendar integration  
- 🧩 Home screen widget (interactive)  
- 🎨 Categories, colors, notes, and task photos  

---

## 🧠 About the App

Most to-do apps help you write tasks.  
Task Planner helps you plan your time.

The app is built with a strong focus on:
- Clean architecture  
- Smooth and responsive UI  
- Local-first data model (no backend)  
- Real-world usability  

---

## 🚀 Tech Stack

- SwiftUI + UIKit  
- SwiftData  
- Combine  
- StoreKit 2  
- WidgetKit + AppIntents  
- UserNotifications & EventKit  
- Swift Charts
